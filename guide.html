A step-by-step reference for hotel teams installing Valpas for the first time. Send this alongside the pre-call email.

Before you start

Prepare everything first

• Download the Valpas app and confirm your account is active.

• Have all hardware ready before starting.

• Choose a router location near a power socket and Ethernet port, with as few walls as possible between the router and the rooms.

iOS: apps.apple.com/us/app/valpas/id1604566976
Android: play.google.com/store/apps/details?id=com.quody.valpasapp

Step 01

Set up the router

Place the router near a power socket and Ethernet port. Keep no more than one wall or ceiling between the router and the rooms. Connect the power cable — the router starts automatically.

Router status lights

🟢 Green — connected and ready.

🔵 Blue or blinking blue — connecting or configuration mode. Wait.

🔴 Red — connection issue. Do not continue. Restart first, then contact OPS if it stays red.

Tip

Install the router first, then move room by room with the legs and cassettes.

Step 02

Install the smart legs and pair the cassettes

Installation steps

1. Find the side of the bed with the headboard or pillows.

2. Install the smart legs on that side of the bed.

3. Make sure the smart legs stay accessible after installation, so the cassette can be removed easily later.

4. If the bed is against a wall, place the smart leg on the side facing the room, not against the wall.

5. Use basic legs in the remaining bed positions.

6. Attach each leg using the threaded rod and the existing bed socket or adapter plate.

7. Make sure each leg is tightened properly and sits straight.

Smart leg rule

Use at least 2 smart legs per bed.

If the bed is for more than 2 sleepers, add 1 more smart leg for each additional sleeper.

Examples

1 sleeper = 2 smart legs

2 sleepers = 2 smart legs

3 sleepers = 3 smart legs

4 sleepers = 4 smart legs

Pairing sequence

1. Profile → Installation Mode → tap +

2. Enter room name → Create

3. Scan QR code → Confirm

Important check

Remove battery strip, reinsert cassette, confirm 3 red blinks. If you see a steady red light, remove and reinsert until it blinks.

Step 03

Check connectivity

When all rooms are installed, open the app and confirm rooms appear online. Rooms should turn green within a few hours.

If some rooms stay offline, add routing devices between the router and the problem rooms. Place one near the router first, then continue outward. Allow up to 8 hours.

If the router goes offline, unplug the power cable, wait 10 seconds, then reconnect it.
